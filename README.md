# Frontend of PIXEL BATTLE #
> (r/place clone)
> Official client: [pixelbattle.com](https://pixelbattle.fun)  
> API: [api.pixelbattle.fun](https://api.pixelbattle.fun/api)  

## How to run? ##
> 1. `npm install`
> 2. `npm run build`
> 3. `npm run serve`

## Stack ##
* Webpack
* Typescript
* Pixi.js
* Pixi-viewport
